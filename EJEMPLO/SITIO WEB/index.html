Northem Design

Este es un Framework CSS de codigo abierto creado para diseñar Sitios Web y Aplicaciones Web de forma rapida. Esta esta basada en los lenguajes HTML y CSS.

Base CSS

Northem se basa en Normalize.css y proporciona diseño y estilo para elementos HTML nativos, además de los componentes de interfaz de usuario más comunes.

Contribuir

Como framework de código abierto, Northem está esperando ver sus valiosas contribuciones. Sientese libre de contribuir en el proyecto atraves de Github.
